# Vantage Red Cogs

This repository contains Vantage-branded cogs for Red-DiscordBot.

## Included Cog

- `vantagemodlog`: detailed server logging with an interactive `/modlog` setup/editor panel.

## Highlights

- First-time setup walkthrough from `/modlog`.
- Interactive controls with channel select menus, multi-selects, buttons, and modals.
- Per-server settings storage.
- Customizable embed style and entry buttons.
- Broad event coverage across messages, moderation, channels, roles, voice, invites, threads, emojis/stickers, and server updates.
